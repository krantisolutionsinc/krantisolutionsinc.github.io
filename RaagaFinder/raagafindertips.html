Raagafinder - Tips and Suggestions

Raagafinder is a web application that enables users to discover the raaga associated with any song. The user plays a song on the synthesizer that is provided by the application. The application then matches the tune played against its database of raagas and recommends a list of raagas that best match the tune.

In order to use Raagafinder, the user is required to have a basic familiarity with using a Piano or a music synthesizer. To make the usage easier, each key in the synthesizer displays the corresponding swara (note) in the Carnatic system of music. The convention followed by Raagafinder to denote the Swaras is described below:

To begin with, there are 12 notes in the Octave. These twelve notes are represented using the 7 basic swaras in Carnatic music:
  1. Shadjamam denoted by S
  2. Rishabam. There are three types:
    1. Suddha Rishabam denoted by R1
    2. Chatusruthi Rishabam denoted by R2
    3. Shatsruthi Rishabam denoted by R3
  3. Gaandharam. There are three types:
    1. Suddha Gaandharam denoted by G1
    2. Saadharana Gaandharam denoted by G2
    3. Anthara Gaandharam denoted by G3
  4. Madhyamam. There are two types:
    1. Suddha Madhyamam denoted by M1
    2. Prathi Madhyamam denoted by M2
  5. Panchamam denoted by P
  6. Daivatham. There are three types:
    1. Suddha Daivatham denoted by D1
    2. Chatusruthi Daivatham denoted by D2
    3. Shatsruthi Daivatham denoted by D3
  7. Nishaadam. There are three types:
    1. Suddha Nishaadam denoted by N1
    2. Kaisika Nishaadam denoted by N2
    3. Kaakali Nishaadam denoted by N3
For those who wonder why the above list of swaras equal 16 when there are only 12 notes in an octave. Well,
  • R2 and G1 are the same frequency.
  • R3 and G2 are the same frequency.
  • D2 and N1 are the same frequency.
  • D3 and N2 are the same frequency.

The Raagafinder maintains a database of about 250 raagas which includes the 72 melagartha raagas and some known janya raagas. We are continuously expanding and verifying our database of raagas. So, please let us know if you feel that a raaga is being reported incorrectly or if you have any other questions or comments.

If you are having problems running the application, please make sure that you install the latest Java Virtual Machine from Sun Microsystems.
